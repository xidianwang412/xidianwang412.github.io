Markdown 入门

发表于 2018-10-06 | 分类于 语言

Markdown

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

基本语法

1. 标题

# 这是一级标题
## 这是二级标题
...
#### 这是四级标题（共支持6级）

2. 列表

Markdown 支持有序列表和无序列表。

无序列表

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，如：

* Red
* Green
* Blue

显示如下：

  • Red
  • Green
  • Blue

有序列表

有序列表使用数字接着一个英文句点，如：

1. Bird
2. McHale
3. Parish

显示如下：

  1. Bird
  2. McHale
  3. Parish

列表项目可以包含多个段落，每个项目下的段落都必须缩进4个空格或是1个制表符

3. 代码区块

和程序相关的写作或是标签语言原始码通常会有已经排版好的代码区块，通常这些区块我们并不希望它以一般段落文件的方式去排版，而是照原来的样子显示。

要在 Markdown 中建立代码区块很简单，只要简单地缩进4个空格或是1个制表符就可以，例如，下面的输入：

这是一个普通段落：

    这是一个代码区块。

显示如下：
这是一个普通段落：

这是一个代码区块。

另外两行(```)之间的区域也是代码区块，如：
```
这里是代码区块
```
显示如下：

4. 强调

Markdown 使用星号（*）和底线（_）作为标记强调字词的符号。

*single asterisks*

_single underscores_

**double asterisks**

__double underscores__

显示如下：
single asterisks

single underscores

double asterisks

double underscores

5. 其他

插入图片

![图片名称](http://gitcafe.com/image.png)
![图片名称](/path/to/img.jpg)

链接

[链接名称](https://github.com)

github

引用

> 第一行引用
> 第二行引用

第一行引用
第二行引用

分隔线

***
或
* * *
或
- - -
等等。

代码

如果要标记一小段行内代码，你可以用反引号把它包起来（`），例如：

Use the `printf()` function.

显示如下：
Use the printf() function.
